📌 Overview

Article Hub is a modern full stack web application designed for creating and managing articles with a secure authentication system.

This project demonstrates real-world development practices, including:

Clean architecture
RESTful API design
Authentication & authorization with JWT
Frontend-backend integration
✨ Features

🔐 Secure authentication (JWT)
📝 Create, edit, delete articles (CRUD)
👤 User registration & login
🛡️ Protected routes (frontend + backend)
⚡ Fast and responsive UI
🌐 RESTful API communication
📊 Scalable backend structure

⚙️ Getting Started
📋 Prerequisites
Node.js
Angular CLI
.NET SDK
SQL Server
📥 Installation
git clone https://github.com/seu-usuario/article-hub.git
cd article-hub
▶️ Run Backend
cd backend
dotnet restore
dotnet run
▶️ Run Frontend
cd frontend
npm install
ng serve
🔐 Authentication Flow
User logs in
Receives JWT token
Sends token in protected requests
Authorization: Bearer YOUR_TOKEN
